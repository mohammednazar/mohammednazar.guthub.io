PDF Splitter

Split a PDF into individual pages and download each page or a single ZIP.

Troubleshooting:
  • If you see “encrypted/password-protected”, unlock the PDF and try again.
  • Ensure the pdf-lib and JSZip scripts load (some extensions can block them).
  • For best results, serve via a local server (e.g., python3 -m http.server 8080).
